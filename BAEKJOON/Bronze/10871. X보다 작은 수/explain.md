아래는 추후 복습을 위해 코드의 작동 원리를, 한 줄씩 “무슨 일이 벌어지는지” 기준으로 자세히 풀어쓴 설명입니다. (with. ChatGPT)

```python
N, X = map(int, input().split())
A = list(map(int, input().split()))

ans = []
i = 0
while i < N:
    if A[i] < X:
        ans.append(str(A[i]))
    i += 1

print(" ".join(ans))
```

---

## 1) 입력 처리 파트

### `N, X = map(int, input().split())`

* `input()` : 한 줄을 문자열로 입력받음. 예) `"10 5"`
* `.split()` : 공백 기준으로 쪼갬. 예) `["10", "5"]`
* `map(int, ...)` : 각 요소를 정수로 변환. 예) `10, 5`
* 최종적으로 `N=10`, `X=5` 같은 형태로 저장됨.

### `A = list(map(int, input().split()))`

* 두 번째 줄 입력 예) `"1 10 4 9 2 3 8 5 7 6"`
* split → 문자열 리스트
* map(int, ...) → 정수로 변환되는 “이터레이터”
* list(...) → 실제 리스트로 고정
* 최종적으로 `A = [1, 10, 4, 9, 2, 3, 8, 5, 7, 6]`

---

## 2) 결과를 담을 준비

### `ans = []`

* **출력할 값(= X보다 작은 값들)**을 담아둘 빈 리스트.
* 왜 바로 출력하지 않고 모으냐?

  * 출력 형식이 “공백으로 이어서 한 줄 출력”이라서,
  * 모았다가 `" ".join(...)`으로 한 번에 찍는 게 편함.

### `i = 0`

* 리스트 `A`를 인덱스로 접근하기 위한 시작점.
* 파이썬 리스트는 0부터 시작하니까 `i=0`이 첫 원소(`A[0]`)를 가리킴.

---

## 3) while 루프의 핵심 동작 원리

### `while i < N:`

* 루프를 “몇 번” 돌릴지 결정하는 조건.
* 의미: `i`가 `0, 1, 2, ..., N-1`일 때만 반복.
* 결과적으로 **A의 0번째부터 N-1번째까지 정확히 한 번씩** 검사함.

즉, 아래처럼 진행됩니다.

* i=0 → A[0] 검사
* i=1 → A[1] 검사
* …
* i=N-1 → A[N-1] 검사
* i=N → 조건 `i < N`이 거짓이므로 종료

---

## 4) 조건 검사 + 정답 리스트에 추가

### `if A[i] < X:`

* 현재 원소 `A[i]`가 기준값 `X`보다 작은지 확인.
* 예를 들어 `X=5`라면,

  * `A[i]=1` → 참(True)
  * `A[i]=10` → 거짓(False)
  * `A[i]=4` → 참(True)
  * …

### `ans.append(str(A[i]))`

* 조건이 참이면 그 값을 `ans`에 넣음.
* 여기서 **str로 바꾸는 이유가 중요**:

  * 마지막에 `" ".join(ans)`를 하려면 **ans 안이 문자열이어야 함**
  * join은 문자열만 이어붙일 수 있음

    * `["1","4","2"]` 가능
    * `[1,4,2]` 불가능(타입 에러)

---

## 5) 인덱스 증가(루프 진행의 핵심)

### `i += 1`

* 이 줄이 없으면 `i`가 계속 0이라서 `A[0]`만 영원히 검사하는 **무한 루프**가 됨.
* 즉, while 루프에서 `i += 1`은 “다음 원소로 넘어가게 만드는 장치”입니다.

---

## 6) 최종 출력

### `print(" ".join(ans))`

* `ans`가 예를 들어 `["1", "4", "2", "3"]`라면
* `" ".join(ans)` → `"1 4 2 3"`
* 문제 요구대로 공백 구분 한 줄로 출력됨.

---

## 예시로 실제 흐름 시뮬레이션

입력:

* N=10, X=5
* A=[1,10,4,9,2,3,8,5,7,6]

반복:

* i=0, A[0]=1  <5 → ans=["1"]
* i=1, A[1]=10 <5 → X → ans 그대로
* i=2, A[2]=4  <5 → ans=["1","4"]
* i=3, A[3]=9  <5 → X
* i=4, A[4]=2  <5 → ans=["1","4","2"]
* i=5, A[5]=3  <5 → ans=["1","4","2","3"]
* i=6, A[6]=8  <5 → X
* i=7, A[7]=5  <5 → 거짓(같으면 제외)
* i=8, A[8]=7  <5 → X
* i=9, A[9]=6  <5 → X
  종료 후 출력: `1 4 2 3`
