백준(BOJ)이나 코딩 테스트에서 파이썬을 사용할 때, **입력 처리**는 가장 기초이자 필수적인 단계입니다. 질문하신 `input()`, `split()`, `map()` 함수는 보통 한 세트로 움직이며, 데이터를 읽고, 자르고, 변환하는 역할을 합니다.

각각의 역할과 이들이 합쳐졌을 때 어떻게 동작하는지 단계별로 설명해 드릴게요.

---

### 1. `input()` : 데이터를 가져오기

사용자가 키보드로 입력한 내용을 **한 줄 단위**로 읽어옵니다.

* **핵심:** 무엇을 입력하든 무조건 **문자열(String)** 로 가져옵니다.
* **예시:**
```python
a = input() 
# 사용자가 10을 입력해도 컴퓨터는 '10'(글자)으로 인식합니다.

```



### 2. `split()` : 데이터를 쪼개기

문자열을 특정 기준에 따라 **잘라서 리스트(List)로** 만듭니다.

* **기본:** 괄호 안을 비우면 **공백(스페이스, 탭, 엔터)** 을 기준으로 자릅니다.
* **예시:**
```python
data = "10 20 30".split()
# 결과: ['10', '20', '30'] 
# (주의: 아직 따옴표가 있는 문자열 상태입니다!)

```



### 3. `map()` : 데이터를 변환하기

리스트 안에 있는 **모든 요소에 똑같은 기능을 한꺼번에 적용**할 때 씁니다.

* **구조:** `map(적용할_함수, 적용할_데이터)`
* **역할:** "문자열 리스트"를 "정수 리스트"로 바꿀 때 핵심적인 역할을 합니다.
* **예시:**
```python
# ['10', '20'] 각각에 int() 함수를 적용해라!
result = map(int, ['10', '20'])
# 결과: 10, 20 (이제 숫자가 되었습니다)

```

### 🚀 백준 실전 조합 (가장 많이 쓰는 패턴 3가지)

백준 문제에서 이 세 가지 함수는 주로 아래와 같은 형태로 결합되어 사용됩니다.

#### 패턴 1: 숫자 두 개를 변수 두 개에 따로 넣을 때

가장 많이 보게 될 형태입니다. (예: `A`와 `B`를 입력받으시오)

```python
# 입력 예시: 10 20
A, B = map(int, input().split())

# 과정 분석:
# 1. input(): "10 20" (문자열)
# 2. split(): ['10', '20'] (문자열 리스트)
# 3. map(int, ...): 10, 20 (정수 변환)
# 4. A, B = ...: A에는 10, B에는 20이 저장됨

```

#### 패턴 2: 숫자들을 리스트(배열)로 한 번에 저장할 때

입력 개수가 많거나 정해지지 않았을 때 사용합니다. (예: `10818번 최소, 최대` 문제 등)
`map` 객체 자체는 리스트가 아니므로, **`list()`** 로 감싸주어야 합니다.

```python
# 입력 예시: 10 20 30 40 50
arr = list(map(int, input().split()))

# 결과: arr = [10, 20, 30, 40, 50]

```

#### 패턴 3: 줄바꿈 없이 붙어있는 숫자를 쪼갤 때

가끔 공백 없이 `12345`처럼 입력되는 경우가 있습니다. 이때는 `split()`을 쓰지 않습니다.

```python
# 입력 예시: 12345
data = list(map(int, input())) # split() 없이 input()만 씀

# 과정: input()은 문자열이므로 '1','2','3','4','5'로 반복 가능(Iterable)함.
# 결과: data = [1, 2, 3, 4, 5]

```

### 💡 꿀팁: 더 빠른 입력을 원한다면?

백준에서 난이도가 올라가면 입력 데이터가 10만 줄, 100만 줄이 넘어갈 때가 있습니다. 이때 `input()`을 쓰면 **시간 초과**가 날 수 있습니다. 그때는 아래 코드를 상단에 적어두고 `input` 대신 `sys.stdin.readline`을 쓰세요.

```python
import sys
input = sys.stdin.readline

# 이후 사용법은 똑같습니다.
a, b = map(int, input().split())

```

(`sys.stdin.readline`은 줄바꿈 문자 `\n`까지 읽어오지만, `split()`을 쓰면 공백과 함께 `\n`도 자동으로 제거되므로 걱정 안 하셔도 됩니다.)

이제 이 함수들의 관계가 좀 명확해지셨나요? 이해를 돕기 위해 **입력받은 두 수의 합을 구하는 간단한 코드**를 직접 작성해보시겠습니까?
